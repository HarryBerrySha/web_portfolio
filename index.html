Virtual Machine

Description

Implemented a high-performance register-based bytecode VM in C.

What I did
  • Designed and shipped a complete runtime (interpreter, assembler, disassembler, verifier) with custom bytecode, constant pool, and native function interface.
  • Built a production-grade memory manager which implemented a mark-and-sweep garbage collector that traces registers, saved call frames, and heap objects, and reclaims memory using an object free-list for stable indices and fast reuse.
  • Added language features including closures and exception handling. Implemented closure objects with captured environments, OP_MK_CLOSURE/OP_CALL_CLOSURE semantics, and try/catch-style handler unwinding that preserves exception values in registers.

  • Optimized call performance and memory safety in C by minimizing call frame overhead by saving only callee clobbered registers, designed array-backed heap for stable object indices, and added comprehensive examples and stress-tests proving capture survival across GC cycles.
